# Amazon MediaStore (amazon-mediastore)
AWS Elemental MediaStore is a video origination and storage service that offers the performance, consistency, and low latency required to deliver live streaming video content. It provides high performance storage with built-in access management for video workloads.

## APIs

### AWS Elemental MediaStore API
The AWS Elemental MediaStore API provides programmatic access to create and manage containers, access policies, CORS policies, and metric policies for video storage and delivery.
